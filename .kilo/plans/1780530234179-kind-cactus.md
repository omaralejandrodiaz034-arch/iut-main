# Plan: Corrección de Indiscponencias en Asignación de Códigos

## PROBLEMA IDENTIFICADO

El archivo `app/Services/CodigoUnicoService.php` tiene la clase nombrada como `CodigoJerarquicoService`, pero los controladores llaman métodos que **NO EXISTEN**:

### Métodos faltantes en CodigoUnicoService/CodigoJerarquicoService:

| Método | Llamado desde | Propósito |
|--------|---------------|-----------|
| `obtenerSiguienteCodigo()` | OrganismoController@create (línea 61), UnidadAdministradoraController@create (línea 90) | Obtener código disponible a nivel sistema |
| `recomendarSiguienteCodigoParaDependencia()` | BienController@recomendarCodigo (línea 231) | Sugerir código para un bien en dependencia específica |
| `codigoExiste()` | OrganismoController, UnidadAdministradoraController, BienController | Verificar si código está en uso |
| `obtenerUbicacionCodigo()` | OrganismoController, UnidadAdministradoraController, BienController | Obtener tabla/nombre donde existe código |
| `reservarCodigosParaOrganismo()` | OrganismoController@store, UnidadAdministradoraController@store | Reservar rango de códigos para unidades |
| `reservarCodigosParaUnidad()` | UnidadAdministradoraController@store | Reservar rango de códigos para dependencias |

### Inconsistencia adicional:
- La validación en `BienController` usa formato de 8 dígitos (`/^\d{8}$/`)
- El servicio genera formato jerárquico (`XXX-XXXX-XXX-XXXXX` con guiones, 19 caracteres)
- **No coinciden** los formatos.

#### A. `obtenerSiguienteCodigo(): string`
```php
public static function obtenerSiguienteCodigo(): string
{
    $ultimoOrganismo = Organismo::orderBy('codigo', 'desc')->first();
    $siguiente = $ultimoOrganismo 
        ? (int)$ultimoOrganismo->codigo + 1 
        : 1;
    
    while (self::codigoExiste(str_pad($siguiente, 8, '0', STR_PAD_LEFT))) {
        $siguiente++;
    }
    
    return str_pad((string)$siguiente, 8, '0', STR_PAD_LEFT);
}
```

#### B. `codigoExiste(string $codigo, ?string $tablaIgnorar = null, ?int $idIgnorar = null): bool`
```php
public static function codigoExiste(string $codigo, ?string $tablaIgnorar = null, ?int $idIgnorar = null): bool
{
    $numero = (int)str_replace('-', '', $codigo);
    
    // Verificar en organismos
    if ($tablaIgnorar !== 'organismos') {
        $exists = Organismo::where('codigo', 'LIKE', "%{$numero}%)->exists();
        if ($exists) return true;
    }
    
    // Verificar en unidades
    if ($tablaIgnorar !== 'unidades') {
        $exists = UnidadAdministradora::where('codigo', 'LIKE', "%{$numero}%)->exists();
        if ($exists) return true;
    }
    
    // Verificar en dependencias
    if ($tablaIgnorar !== 'dependencias') {
        $exists = Dependencia::where('codigo', 'LIKE', "%{$numero}%)->exists();
        if ($exists) return true;
    }
    
    // Verificar en bienes
    if ($tablaIgnorar !== 'bienes') {
        $exists = Bien::where('codigo', 'LIKE', "%{$numero}%)->where('id', '!=', $idIgnorar)->exists();
        return $exists;
    }
    
    return false;
}
```

#### C. `obtenerUbicacionCodigo(string $codigo): array`
```php
public static function obtenerUbicacionCodigo(string $codigo): array
{
    $numero = (int)str_replace('-', '', $codigo);
    
    $organismo = Organismo::where('codigo', 'LIKE', "%{$numero}%")->first();
    if ($organismo) {
        return ['tabla' => 'organismos', 'nombre' => $organismo->nombre];
    }
    
    $unidad = UnidadAdministradora::where('codigo', 'LIKE', "%{$numero}%")->first();
    if ($unidad) {
        return ['tabla' => 'unidades_administradoras', 'nombre' => $unidad->nombre];
    }
    
    $dependencia = Dependencia::where('codigo', 'LIKE', "%{$numero}%")->first();
    if ($dependencia) {
        return ['tabla' => 'dependencias', 'nombre' => $dependencia->nombre];
    }
    
    $bien = Bien::where('codigo', 'LIKE', "%{$numero}%")->first();
    if ($bien) {
        return ['tabla' => 'bienes', 'nombre' => $bien->descripcion];
    }
    
    return ['tabla' => null, 'nombre' => null];
}
```

#### D. `recomendarSiguienteCodigoParaDependencia(int $dependenciaId): array`
```php
public static function recomendarSiguienteCodigoParaDependencia(int $dependenciaId): array
{
    return DB::transaction(function () use ($dependenciaId) {
        $dependencia = Dependencia::with(['unidadAdministradora.organismo'])
            ->lockForUpdate()
            ->findOrFail($dependenciaId);
        
        $siguiente = max(
            $dependencia->code_min,
            Bien::where('dependencia_id', $dependenciaId)->max('codigo') + 1
        );
        
        if ($siguiente > $dependencia->code_max) {
            throw new RuntimeException(
                "Rango de códigos exhausto. Rango: {$dependencia->code_min}-{$dependencia->code_max}"
            );
        }
        
        $orgCode = str_pad($dependencia->unidadAdministradora->organismo->codigo, 3, '0', STR_PAD_LEFT);
        $uaCode = str_pad($dependencia->unidadAdministradora->codigo, 4, '0', STR_PAD_LEFT);
        $depCode = str_pad($dependencia->codigo, 3, '0', STR_PAD_LEFT);
        $secCode = str_pad($siguiente, 5, '0', STR_PAD_LEFT);
        
        return [
            'codigo' => "{$orgCode}-{$uaCode}-{$depCode}-{$secCode}",
            'siguiente_numero' => $siguiente,
            'rango_min' => $dependencia->code_min,
            'rango_max' => $dependencia->code_max,
            'disponibles_restantes' => $dependencia->code_max - $siguiente + 1,
        ];
    });
}
```
