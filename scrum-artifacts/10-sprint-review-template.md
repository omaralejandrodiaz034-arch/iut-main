# Sprint Review Template

## Sistema de Gestión de Inventario de Bienes

> **Duración:** 2 horas  
> **Participantes:** Equipo Scrum + Product Owner + Stakeholders

---

## Agenda

| Tiempo | Actividad |
|--------|-----------|
| 0-15 min | Bienvenida y agenda |
| 15-45 min | Demostración de funcionalidades |
| 45-75 min | Feedback del Product Owner |
| 75-90 min | Actualización de backlog |
| 90-120 min | Próximos pasos |

---

## Demo Checklist

```markdown
## Demo - Sprint [X] - [Fecha]

### Historias Demostradas

| ID | Historia | Demo | Feedback |
|----|----------|------|----------|
| HU-XXX | [Nombre] | ✅ / ⏳ | [Comentarios] |

### Funcionalidades Mostradas

1. [ ] Login/Logout
2. [ ] CRUD de bienes
3. [ ] Movimientos
4. [ ] Reportes PDF
5. [ ] Dashboard
6. [ ] [Nuevas funcionalidades]
```
