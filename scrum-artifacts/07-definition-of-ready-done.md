# Definition of Ready (DoR) y Definition of Done (DoD)

## Sistema de Gestión de Inventario de Bienes

---

## Definition of Ready (DoR)

Una historia de usuario está lista para ser incluida en un sprint cuando cumple los siguientes criterios:

### Criterios Obligatorios

| # | Criterio | ✅ |
|---|----------|-----|
| 1 | **Título claro** | Historia tiene nombre descriptivo |
| 2 | **Descripción completa** | Formato "Como [rol], quiero [funcionalidad], para [beneficio]" |
| 3 | **Épica identificada** | Historia asignada a épica del proyecto |
| 4 | **Criterios de aceptación** | Mínimo 3 criterios medibles y verificables |
| 5 | **Estimación asignada** | Puntos de historia usando escala Fibonacci |
| 6 | **Dependencias identificadas** | Cualquier dependencia documentada |
| 7 | **Mockups o wireframes** | Para historias con componente UI significativo |
| 8 | **Datos de prueba definidos** | Escenarios de prueba identificados |

---

## Definition of Done (DoD)

Una historia de usuario se considera terminada cuando:

### Criterios Técnicos

| # | Criterio | ✅ |
|---|----------|-----|
| 1 | **Código implementado** | Según criterios de aceptación |
| 2 | **Code review aprobado** | Revisado por al menos un compañero |
| 3 | **Tests unitarios pasando** | Cobertura mínima 70% |
| 4 | **Tests de integración pasando** | Funcionalidad probada en entorno local |
| 5 | **Sin bugs críticos/alta** | Issues resueltos |
| 6 | **Migraciones ejecutadas** | BD actualizada correctamente |
| 7 | **Sin warnings/lint errors** | Código pasa validaciones |

### Criterios Funcionales

| # | Criterio |
|---|----------|
| 8 | **Criterios de aceptación cumplidos** |
| 9 | **Pruebas en desarrollo** |
| 10 | **Documentación actualizada** |

### Criterios de Calidad

| # | Criterio |
|---|----------|
| 11 | **Accesibilidad básica** |
| 12 | **Diseño responsive** |
| 13 | **Rendimiento aceptable** |

### Criterios de Proceso

| # | Criterio |
|---|----------|
| 14 | **Aprobación del Product Owner** |
| 15 | **Código mergeado** |
| 16 | **Deployment preparado** |
