# Daily Standup Template

## Sistema de Gestión de Inventario de Bienes

> **Duración:** 15 minutos  
> **Horario:** 9:00 AM diario  
> **Lugar:** #scrum-standup (Slack/Discord) o presencial

---

## Formato Estándar

Cada miembro del equipo responde estas 3 preguntas:

### 1️⃣ ¿Qué hice ayer?

```markdown
- [Historia ID]: [actividad específica]
- Ejemplo: HU-022: Implementé validación de columnas Excel
```

### 2️⃣ ¿Qué haré hoy?

```markdown
- [Historia ID]: [tarea planificada]
- Ejemplo: HU-022: Crear reporte de errores de importación
```

### 3️⃣ ¿Impedimentos?

```markdown
- [Sí/No]: [descripción breve]
- Si es urgente: @ScrumMaster
```
